---
<a id="installing"></a>
## Installing for termux

* `termux-setup-storage`
* `apt full-upgrade`
* `apt install git`
* `git clone https://github.com/d4rklov3r/mail`
* `cd mail`
* `bash install.sh`

## Installing for linux

* `apt upgrade`
* `apt install git`
* `git clone https://github.com/d4rklov3r/mail`
* `cd mail`
* `bash install.sh`

## Start Gmail-Hack

* `cd && cd mail && python3 mail.py`
#### If it does not work then the command:
* `python3 mail.py`
